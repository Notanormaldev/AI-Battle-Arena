## вҡ”пёҸ What is AI Battle Arena?

**AI Battle Arena** is a real-time platform where multiple large language models compete against each other on the same prompt вҖ” simultaneously. You pick the question. The AIs fight for the best answer. You decide who wins.

No benchmarks. No synthetic tests. Just raw, unfiltered AI intelligence вҖ” judged by you.

## рҹ§  The Intelligence Stack

At its core, AI Battle Arena is built on a **stateful multi-agent orchestration pipeline** powered by **LangGraph** вҖ” a graph-based execution engine that coordinates multiple AI agents with full control over state, routing, and parallelism.

**LangChain** acts as the universal adapter layer, standardizing communication across wildly different AI providers so every model gets the same prompt, under the same conditions, with zero bias.

## рҹ”¬ How LangGraph Powers the Battles

LangGraph enables something standard API calls can't вҖ” **stateful, parallel, conditional multi-agent execution**.

Each battle is a **compiled graph** with discrete nodes:

- **Dispatch Node** вҖ” Fans out the prompt to all registered model nodes in parallel
- **Model Nodes** вҖ” Each AI provider runs as an isolated node with its own retry logic and timeout handling
- **Aggregation Node** вҖ” Collects all responses, normalizes them via LangChain's unified schema, and prepares the battle result
- **State** вҖ” The entire battle lifecycle (prompt вҶ’ responses вҶ’ metadata) flows through a typed state object, enabling reproducibility and debugging

This graph-based approach means adding a new AI model is as simple as adding a new node вҖ” the orchestration logic stays untouched.
